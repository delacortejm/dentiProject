# 🦷 Sistema de Gestión Dental v2.0

Sistema profesional de gestión para consultorios odontológicos desarrollado en Streamlit.

## Características
- Dashboard interactivo
- Calculadora de precios profesional
- Benchmarks oficiales del sector
- Análisis de costos integral
- Planificación de objetivos

## Instalación
```bash
pip install -r requirements.txt
streamlit run app.py
